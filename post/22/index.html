Manjaro 平台安装 hugo

sudo pacman -S hugo

建立 hugo 项目

hugo new site [project-name]

添加主题

为了加速博客搭建, 我们可以使用主题,这里以 m10c 样式为例. hugo 主题样式

git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

启动 hugo

hugo server -t m10c –buildDrafts 用 -t 参数制定主题样式.

添加新的博客

hugo new post/firstBlog.md

部署到 github

建立 github 仓库

  • 仓库名称 username.github.io

发布到 github

  • 生成静态网页 hugo –theme=even –baseUrl="https://yourname.github.io”
  • 进入 public 目录.
  • git add . 添加当前目录下的所有文件.
  • git commit 提交文件.
  • 同步 github 仓库. git push -u origin master.
